# Electron Pro

### 基于 [Umi](https://umijs.org/) 和 [TypeScript](https://www.typescriptlang.org/) 的 [Electron](https://www.electronjs.org/) 模板

## 安装

通过 git 克隆仓库到本地

```shell
$ git clone https://github.com/kit48/electron-pro.git
```

安装依赖

```shell
$ npm install
# or
$ yarn
```

## 开发

```shell
$ npm start
```

## 打包

```shell
$ npm run pack
```

如果想打包成一个 dmg 或 zip 文件，可以以下命令实现

```shell
$ npm run dist
```
